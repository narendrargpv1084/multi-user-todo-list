About

The To-Do List App is a simple and user-friendly web application that helps you organize and manage your daily tasks effectively.

Each user has their own private To-Do list. You can add new tasks, edit them inline, update their status (Pending / Completed), and remove tasks.

Features:

  • Multi-user support
  • Personal To-Do list per user
  • Login / Signup functionality
  • Persistent data storage (LocalStorage)
  • Responsive design (Mobile / Desktop friendly)
  • Inline editing of tasks

Created by Narendra Singh Rajput as a simple personal project to practice web development skills.
